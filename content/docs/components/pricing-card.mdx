import {
  PricingCard,
  PricingCardHeader,
  PricingCardBadge,
  PricingCardTitle,
  PricingCardDescription,
  PricingCardPrice,
  PricingCardFeatures,
  PricingCardFeature,
  PricingCardAction,
} from "@/components/billing";

A composable pricing card component built with compound components pattern. Perfect for displaying subscription plans, pricing tiers, and feature comparisons.

## Installation

```bash
npx shadcn@latest add pricing-card --from https://billui.com/api/registry
```

## Usage

```tsx
import {
  PricingCard,
  PricingCardHeader,
  PricingCardBadge,
  PricingCardTitle,
  PricingCardDescription,
  PricingCardPrice,
  PricingCardFeatures,
  PricingCardFeature,
  PricingCardAction,
} from "@/components/ui/pricing-card";
```

```tsx
<PricingCard>
  <PricingCardHeader>
    <PricingCardTitle>Pro</PricingCardTitle>
    <PricingCardDescription>For growing teams</PricingCardDescription>
  </PricingCardHeader>
  <PricingCardPrice amount={29} period="month" />
  <PricingCardFeatures>
    <PricingCardFeature included>Unlimited projects</PricingCardFeature>
    <PricingCardFeature included>Priority support</PricingCardFeature>
    <PricingCardFeature included={false}>Custom integrations</PricingCardFeature>
  </PricingCardFeatures>
  <PricingCardAction>Get Started</PricingCardAction>
</PricingCard>
```

## API Reference

### PricingCard

The root container component.

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `variant` | `"default" \| "highlighted" \| "compact"` | `"default"` | Visual style variant |
| `size` | `"default" \| "lg"` | `"default"` | Size variant |
| `className` | `string` | - | Additional CSS classes |

### PricingCardHeader

Container for the title, description, and badge.

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `className` | `string` | - | Additional CSS classes |

### PricingCardBadge

Optional badge for highlighting plans.

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `variant` | `"default" \| "secondary" \| "outline"` | `"default"` | Visual style variant |
| `className` | `string` | - | Additional CSS classes |

### PricingCardTitle

The plan name/title.

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `className` | `string` | - | Additional CSS classes |

### PricingCardDescription

Short description of the plan.

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `className` | `string` | - | Additional CSS classes |

### PricingCardPrice

Displays the price with optional original price for discounts.

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `amount` | `number` | Required | The price amount |
| `currency` | `string` | `"$"` | Currency symbol |
| `period` | `"month" \| "year" \| "once"` | `"month"` | Billing period |
| `originalAmount` | `number` | - | Original price (for showing discounts) |
| `className` | `string` | - | Additional CSS classes |

### PricingCardFeatures

Container for the feature list.

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `className` | `string` | - | Additional CSS classes |

### PricingCardFeature

Individual feature item with included/excluded state.

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `included` | `boolean` | `true` | Whether the feature is included |
| `className` | `string` | - | Additional CSS classes |

### PricingCardAction

CTA button for the pricing card.

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `variant` | `"default" \| "outline" \| "secondary"` | `"default"` | Button style variant |
| `className` | `string` | - | Additional CSS classes |
