## Overview

Payment Form provides composable layout components that wrap any payment provider (Stripe Elements, Braintree Hosted Fields) or native inputs. billui controls layout, styling, and accessibility—you control what goes inside.

### What billui controls

- Layout and spacing
- Labels and typography
- Error, loading, and disabled states
- Focus rings and borders
- Accessibility (aria attributes, focus management)

### What you control

- The payment provider (Stripe, Braintree, native, etc.)
- Payment logic and tokenization
- Form submission handling

## Usage with Stripe Elements

The components are designed to work seamlessly with Stripe Elements:

```tsx
import { CardElement } from "@stripe/react-stripe-js";
import {
  PaymentForm,
  PaymentFormSection,
  PaymentFormField,
  PaymentFormSubmit,
} from "@/components/ui/payment-form";

function CheckoutForm() {
  const [error, setError] = useState<string>();

  return (
    <PaymentForm onSubmit={handleSubmit}>
      <PaymentFormSection>
        <PaymentFormField label="Card details" error={error}>
          <CardElement
            onChange={(e) => setError(e.error?.message)}
            options={{
              style: {
                base: {
                  fontSize: "16px",
                  color: "hsl(var(--foreground))",
                  "::placeholder": { color: "hsl(var(--muted-foreground))" },
                },
              },
            }}
          />
        </PaymentFormField>
      </PaymentFormSection>
      <PaymentFormFooter>
        <PaymentFormSubmit>Pay</PaymentFormSubmit>
      </PaymentFormFooter>
    </PaymentForm>
  );
}
```

## API Reference

### PaymentForm

The root form container.

| Prop | Type | Default |
| --- | --- | --- |
| `onSubmit` | `(e: FormEvent) => void` | — |

### PaymentFormSection

Groups related fields with a card-like container.

| Prop | Type | Default |
| --- | --- | --- |
| `variant` | `"default" \| "elevated"` | `"default"` |

### PaymentFormField

Wrapper for form fields with label and error support.

| Prop | Type | Default |
| --- | --- | --- |
| `label` | `string` | — |
| `name` | `string` | — |
| `error` | `string` | — |
| `description` | `string` | — |
| `required` | `boolean` | `false` |
| `asChild` | `boolean` | `false` |

### PaymentFormRow

Horizontal layout for multiple fields.

| Prop | Type | Default |
| --- | --- | --- |
| `columns` | `2 \| 3 \| 4` | `2` |

### PaymentFormSubmit

Submit button with loading state.

| Prop | Type | Default |
| --- | --- | --- |
| `loading` | `boolean` | `false` |
| `showSecureIcon` | `boolean` | `true` |
