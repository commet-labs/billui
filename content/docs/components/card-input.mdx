## Overview

Card Input provides native HTML inputs with card-specific formatting, validation, and brand detection. Use these for:

- **Demos and prototypes** - Professional-looking card forms without a payment provider
- **Custom PSP integrations** - When your PSP allows direct card input
- **Testing** - Quickly test checkout flows

### Features

- Auto-formatting as you type (4242 4242 4242 4242)
- Real-time card brand detection (Visa, Mastercard, Amex, Discover, etc.)
- Dynamic brand icon
- Expiry date mask (MM/YY)
- CVC length based on card brand (3 for most, 4 for Amex)
- Proper `inputMode="numeric"` for mobile keyboards

<Callout type="warning">
**PCI Compliance Note**: These inputs send card data through your server. For production use with real cards, use a PCI-compliant provider like Stripe Elements or Braintree Hosted Fields wrapped in [Payment Form](/docs/components/payment-form).
</Callout>

### Controlled Inputs

```tsx
import { useState } from "react";
import {
  CardInputGroup,
  CardNumberInput,
  CardExpiryInput,
  CardCvcInput,
  type CardInputBrand,
} from "@/components/ui/card-input";

function CardForm() {
  const [cardNumber, setCardNumber] = useState("");
  const [expiry, setExpiry] = useState("");
  const [cvc, setCvc] = useState("");
  const [brand, setBrand] = useState<CardInputBrand>("unknown");

  return (
    <CardInputGroup>
      <CardNumberInput
        value={cardNumber}
        onChange={setCardNumber}
        onBrandChange={setBrand}
      />
      <CardExpiryInput value={expiry} onChange={setExpiry} />
      <CardCvcInput value={cvc} onChange={setCvc} brand={brand} />
    </CardInputGroup>
  );
}
```

| Discover | 6011, 644-649, 65 |
| Diners Club | 36, 38, 300-305 |

## API Reference

### CardInputGroup

Container that combines all card inputs with proper styling.

### CardNumberInput

Formatted card number input with brand detection.

| Prop | Type | Default |
| --- | --- | --- |
| `value` | `string` | `""` |
| `onChange` | `(value: string) => void` | — |
| `onBrandChange` | `(brand: CardBrand) => void` | — |
| `showBrandIcon` | `boolean` | `true` |
| `placeholder` | `string` | `"1234 1234 1234 1234"` |

### CardExpiryInput

Expiry date input with MM/YY masking.

| Prop | Type | Default |
| --- | --- | --- |
| `value` | `string` | `""` |
| `onChange` | `(value: string) => void` | — |
| `placeholder` | `string` | `"MM/YY"` |

### CardCvcInput

CVC/CVV input with length based on card brand.

| Prop | Type | Default |
| --- | --- | --- |
| `value` | `string` | `""` |
| `onChange` | `(value: string) => void` | — |
| `brand` | `CardBrand` | `"unknown"` |
| `placeholder` | `string` | `"CVC"` |

### Utility Functions

```tsx
import {
  detectCardBrand,
  formatCardNumber,
  formatExpiry,
  getCardMaxLength,
  getCvcMaxLength,
} from "@/components/ui/card-input";

// Detect brand from number
detectCardBrand("4242"); // "visa"

// Format card number
formatCardNumber("4242424242424242", "visa"); // "4242 4242 4242 4242"

// Format expiry
formatExpiry("1225"); // "12/25"

// Get max card digits (15 for Amex, 16 for others)
getCardMaxLength("amex"); // 15

// Get CVC length (4 for Amex, 3 for others)
getCvcMaxLength("amex"); // 4
```

type CardBrand =
