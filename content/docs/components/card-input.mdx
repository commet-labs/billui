## API Reference

### CardInputGroup

Container that groups card inputs together with shared focus states and border styling.

| Prop | Type | Default |
| --- | --- | --- |
| `error` | `boolean` | `false` |

### CardNumberInput

Card number input with automatic formatting, brand detection, and Luhn validation.

| Prop | Type | Default |
| --- | --- | --- |
| `value` | `string` | — |
| `onChange` | `(value: string) => void` | — |
| `onBrandChange` | `(brand: CardBrand) => void` | — |
| `onValidationChange` | `(validation: { isValid: boolean; isComplete: boolean }) => void` | — |
| `showBrandIcon` | `boolean` | `true` |
| `showValidation` | `boolean` | `true` |
| `placeholder` | `string` | `"1234 1234 1234 1234"` |

### CardExpiryInput

Expiry date input with MM/YY formatting and expiration validation.

| Prop | Type | Default |
| --- | --- | --- |
| `value` | `string` | — |
| `onChange` | `(value: string) => void` | — |
| `onValidationChange` | `(validation: { isValid: boolean; isComplete: boolean; isExpired: boolean }) => void` | — |
| `showValidation` | `boolean` | `true` |
| `placeholder` | `string` | `"MM/YY"` |

### CardCvcInput

CVC/CVV input that adapts to card brand (3 digits for most cards, 4 for Amex).

| Prop | Type | Default |
| --- | --- | --- |
| `value` | `string` | — |
| `onChange` | `(value: string) => void` | — |
| `brand` | `CardBrand` | `"unknown"` |
| `placeholder` | `string` | `"CVC"` |

### CardBrand

Type representing supported card brands.

```tsx
type CardBrand =
  | "visa"
  | "mastercard"
  | "amex"
  | "discover"
  | "diners"
  | "jcb"
  | "unionpay"
  | "unknown";
```

### Utility Functions

The component exports several utility functions for custom implementations:

| Function | Description |
| --- | --- |
| `detectCardBrand(cardNumber: string)` | Returns the card brand based on the card number prefix |
| `validateLuhn(cardNumber: string)` | Validates a card number using the Luhn algorithm |
| `validateCardNumber(cardNumber: string)` | Returns `{ isValid, isComplete, brand }` for a card number |
| `validateExpiry(expiry: string)` | Returns `{ isValid, isComplete, isExpired }` for an expiry date |
| `formatCardNumber(value: string, brand: CardBrand)` | Formats a card number with proper spacing |
| `formatExpiry(value: string)` | Formats an expiry date as MM/YY |
| `getCardMaxLength(brand: CardBrand)` | Returns max digits for a card brand (15 for Amex, 16 for others) |
| `getCvcMaxLength(brand: CardBrand)` | Returns CVC length for a card brand (4 for Amex, 3 for others) |
