## Features

- **Spring physics**: All animations use spring physics for natural, fluid motion
- **Animated progress bars**: Progress indicators animate smoothly when values change
- **Animated numbers**: Values animate when they change using spring interpolation
- **Staggered list items**: Items animate in sequence when expanding/collapsing
- **Smooth expand/collapse**: List expansion uses AnimatePresence for seamless transitions
